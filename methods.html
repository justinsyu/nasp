How This Page Was Built

Extraction rule

Methods are counted only when rule-based text evidence appears in the parsed poster markdown. Records without explicit method language are classified as no explicit statistical method reported.

Traceability rule

Each result links to the saved PDF and markdown. Evidence snippets are stored on the individual poster record so counts can be inspected rather than treated as untraceable summaries.

Sample-size rule

The sample-size bucket uses the largest detected analytic denominator in the parsed text. The original snippet is retained because posters often report multiple denominators.

Outcome-variable rule

Outcome variables are extracted only from explicit outcome, endpoint, measure, evaluation, methods, or results language. Background claims and future/not-assessed outcomes are excluded.
