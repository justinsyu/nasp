Objective

Translate routine specialty pharmacist work into an economic signal.

The poster's aim was to estimate the value of pharmacist interventions in a specialty pharmacy setting and clarify what high-touch specialty pharmacy care can contribute to patients and the health care system.

Evidence flow

How the cost-avoidance estimate was built

Code interventions

A workflow categorized pharmacist interventions and identified higher-likelihood cost-saving "Impacts."

Review impacts

An investigator reviewed a subset of documented impacts to assign likely prevented outcomes.

Attach dollars

Prevented medical and prescription costs were estimated using standardized assumptions and inflation adjustment where needed.

Run scenarios

Annual value was reported for all prevented outcomes and a conservative 50% prevented-outcome scenario.

Limitations

Interpret as an estimate, not a precise causal savings claim.

  • A single pharmacist reviewed impacts and assigned associated avoided outcomes.
  • Hospitalization cost inputs were standardized but came from an older database.
  • The prevented outcome attached to an intervention required subjective judgment.

Conclusion

Specialty pharmacist interventions produced meaningful estimated avoided cost.

Using the poster's assumptions, routine specialty pharmacy care was associated with an estimated $174.38 to $318.75 in value per intervention, while the small number of flagged impacts likely underrepresented all cost-saving activity.
